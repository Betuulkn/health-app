# Health App - BMI & BMR Calculator
# Fonctionnalités
	- Calcul du BMI
	- Calcul du BMR
	- API REST 
	- Intégration avec Github actions
#Prérequis
	- Python 3.9 ou supérieur
	- Docker
	- Git
	- Azure CLI
#Installation
Cloner le dépôt Github : 
	git clone https://github.com/Betuulkn/health-app.git
	cd health-app
